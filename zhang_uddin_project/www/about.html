About

Welcome to the Vanguard Deck Builder! This is a supplement to help with creating decks for the physical trading card game, Cardfight!! Vanguard. Information about all options of the application is below. Click on the list to jump to a section.

  • Browse
  • Build
  • View Deck
  • Edit Deck
  • Playtest
  • Profile

Browse/All Decks

By selecting "Browse Decks" in the side menu, you can search for decks other people have created. You can search by deck name, main clan, or the deck creator. If the deck name or creator box is left blank, they will be treated as "Any." Then press the "Search Decks" button to display the decks.

Decks will be displayed with their deck name, their clan name and symbol, their creator, and their average rating. Clicking on the creator's name will bring you to the creator's Profile page. Pressing the View Deck button will bring you to a page where you can view the cards in the deck.

Build/Your Decks

By selecting "Build Decks" in the side menu, you can view all decks you have created. You can add a new deck by using the form at the top. Type a deck name, deck description (optional), and a main clan, and press the "Add Deck" button. The deck will now be available to edit and be viewable by others. Keep in mind that the main clan is mainly symbolic and cards from other clans can still be added to the deck.

Decks are displayed by their deck name, clan name and image, creator (you), and average rating. You have buttons to View Decks, Edit Decks by adding cards, and delete decks.

View Deck

This is the page that displays a deck's contents. This pages shows the deck name, the creator, and the description. In the "Rate" section, you can post a rating from 1 to 5 for the deck. The "Show Decklist" button will provide a quick overview list of all the cards in the deck, grouped by grade. The "Draw Starting Hand" button will take you to a Playtest section, where you can see sample starting hands for the deck. The image of each card in the deck is shown at the bottom, sorted by grade. Clicking on an image will make it bigger. If you are the deck creator, you will also have the option to Edit the Deck and remove cards with the "Remove Card" button.

Edit Deck/Browse Cards

This page allow you to edit your decks. After you have selected a deck to edit, you will be brought to a search page that allows you to browse through the card pool. To add a card, first search for it by using the search form at the top. Enter a card name, a clan, and a grade, and press the "Search Cards" button to display the cards that match the criteria. If card name is left blank, it is treated as "Any."

Cards will be displayed with their card art, name, clan, and grade. Clicking on a image will make it bigger. To add a card, select the number to add, and press the "Add Cards" button. An alert will display, showing how many copies of the card were added to the deck. To go directly to the deck, press the deck name displayed at the top of the page.

Playtest

This page can be accessed from the "Draw Starting Cards" button under View Deck pages, it displays a five card starting hand for a deck. This means the deck is shuffled and the first five cards are displayed by their image and name. Clicking an image will make it bigger. Another test hand can be drawn by pressing "Draw Another Hand" OR by shaking your device.

Profile

User profiles can be accessed by pressing "Profile" on the side menu or pressing on a user's name when viewing decks. The profile page displays the name of the user, the user's profile picture, and all of the user's decks, listed by deck name, clan, average rating, and with a button to View the Deck.

If you access the Profile page from the side menu or by clicking on your username, you will have buttons to "Take Picture" and "Upload Picture" under the profile picture. "Take Picture" opens the camera to capture a profile picture, and "Upload Picture" opens the gallery to post an already taken picture as the profile picture. You will also have a button to Edit Deck under each deck.
